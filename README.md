# 项目概述 java541

本项目是一个知识管理系统，主要用于搭建属于个人或团队的 WIKI 系统。该系统涵盖了源码和数据库 SQL，并提供了详细的运行说明。

## 功能模块

系统的主要功能包括：

- 在线管理公司、项目文档以及任意形式的文档资料。
- 文档空间管理，支持空间划分，人员分组授权。
- 支持文档空间的收藏，以及空间内文档的开放访问。
- 文档编辑、查看、评论、上传附件。
- 历史版本查看，页面权限控制，文档搜索。
- 文档编辑支持 HTML 富文本和 Markdown 两种方式。
- 统一管理 Swagger 文档、OpenApi 文档及自建接口文档。
- 文档查看、接口请求、全局参数管理等功能。

## 系统角色

- 系统管理员：负责系统整体的配置和管理。
- 普通用户：可以创建、编辑、查看和管理个人或团队文档。

## 运行环境

- 后端：Java 1.8 或以上版本，兼容主流数据库如 MySQL。
- 前端：现代浏览器。

## 部署步骤

1. 准备后端运行环境。
2. 导入数据库 SQL 文件。
3. 部署后端服务。
4. 部署前端项目。
